## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/vanh2604/quizlet-clone.git
   ```
2. Create a iOS Simulator device using [Xcode](https://developer.apple.com/xcode/) or AVD using [Android Studio](https://developer.android.com/studio)
3. Run the app
   ```
   flutter run
   ```
